Pusat Bantuan

Cara Top Up Diamond

Ikuti langkah-langkah mudah untuk melakukan top up diamond di DepsStore:

  1. Pilih game yang ingin Anda top up
  2. Masukkan ID game dan server
  3. Pilih jumlah diamond yang diinginkan
  4. Pilih metode pembayaran
  5. Selesaikan pembayaran
  6. Tunggu diamond masuk ke akun game Anda (maksimal 15 menit)

Panduan Troubleshooting

Jika mengalami masalah, coba langkah-langkah berikut:

  • Pastikan ID game dan server sudah benar
  • Periksa koneksi internet Anda
  • Refresh halaman jika proses terhenti
  • Hubungi customer service jika diamond tidak masuk dalam 30 menit
  • Simpan ID transaksi untuk referensi

Panduan Pembayaran

Langkah-langkah Pembayaran:

  1. Pilih metode pembayaran yang diinginkan
  2. Ikuti instruksi yang muncul di layar
  3. Untuk e-wallet: pastikan saldo mencukupi
  4. Untuk QRIS: scan kode yang muncul menggunakan aplikasi bank/e-wallet
  5. Tunggu konfirmasi pembayaran
  6. Simpan bukti transaksi

Perhatian:

  • Jangan tutup halaman selama proses pembayaran
  • Pastikan pembayaran telah berhasil sebelum menutup halaman
  • Hubungi customer service jika terjadi kegagalan transaksi

Pertanyaan Umum (FAQ)

Berapa lama waktu yang dibutuhkan untuk diamond masuk ke akun game?

Biasanya diamond akan masuk ke akun game Anda dalam waktu 1-5 menit. Namun, dalam beberapa kasus bisa memakan waktu hingga 15 menit. Jika diamond belum masuk setelah 30 menit, silakan hubungi customer service dengan menyertakan ID transaksi.

Apa yang harus dilakukan jika diamond tidak masuk ke akun game?

Pertama, pastikan ID game dan server yang dimasukkan sudah benar. Jika sudah benar, silakan hubungi customer service dengan menyertakan bukti pembayaran dan ID transaksi. Tim kami akan memproses keluhan Anda dalam waktu 1x24 jam.

Bagaimana cara mengetahui ID transaksi?

ID transaksi akan dikirimkan ke email Anda setelah pembayaran berhasil. Anda juga dapat melihatnya di halaman riwayat transaksi pada akun DepsStore Anda. ID transaksi biasanya dimulai dengan "TRX-" diikuti oleh angka.

Apakah DepsStore menjamin keamanan data pribadi saya?

Ya, DepsStore sangat menjaga keamanan dan privasi data pelanggan. Semua data pribadi dan transaksi dienkripsi dengan teknologi terkini. Kami tidak akan membagikan data Anda kepada pihak ketiga tanpa izin.

Apakah ada biaya tambahan untuk transaksi di DepsStore?

Tidak, harga yang tertera di DepsStore adalah harga final. Tidak ada biaya tambahan atau biaya administrasi apapun. Namun, beberapa metode pembayaran mungkin mengenakan biaya sesuai kebijakan penyedia pembayaran tersebut.
